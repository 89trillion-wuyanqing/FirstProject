

## 1.第一个接口

### 1.接口名称

GetArmys()

### 2.接口功能

根据输入的稀有度参数、当前解锁阶段参数和cvc参数，获取符合稀有度、cvc且已解锁的所有士兵数据并返回。

### 3.URL

[http:127.0.0.1:8000/getArmys](http:127.0.0.1:8000/getArmys)

### 4.HTTP请求方式

POST

### 5.请求参数

| 参数        | 必选 | 类型 | 说明             |
| ----------- | ---- | ---- | ---------------- |
| rarity      | true | int  | 稀有度           |
| unlockArena | true | int  | 当前解锁阶段参数 |
| cvc         | true | int  | cvc              |

### 6.返回字段

| 返回字段 | 字段类型 | 说明         |
| -------- | -------- | ------------ |
| code     | int      | 状态码       |
| message  | string   | 请求状态消息 |
| data     | json     | 返回信息     |





### 7.接口示例

[http:127.0.0.1:8000/getArmys](http:127.0.0.1:8000/getArmys)

## 命令行

在项目执行可执行文件的执行命令

```sh
./main -f="conf/config.army.model.json"
```

在app目录下执行main.go时需要设置working directory为项目根路径



在单元测试代码目录下执行go test时设置working directory为项目根路径
